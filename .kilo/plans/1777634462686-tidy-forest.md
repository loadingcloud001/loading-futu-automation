# US Options Trading Notion 自動化方案

## 架構

```
futu-opend Docker (Futu API)
  │
  ▼
option-python Docker (app.py)
  │
  ├─→ Google Sheets: Today/Yesterday (保留，向後相容)
  ├─→ CSV: 本地備份
  └─→ Notion: 自動寫入 + 保留歷史
        │
        ├── 📊 Daily Snapshot   (今日快照，每日清除重建)
        ├── 📈 Historical Archive (長期資料庫，每日附加)
        └── 📝 Trade Journal     (交易日誌，使用者手動填寫)
```

### 每日自動化流程

```
04:00 UTC 觸發
  │
  ├─ 1. Futu OpenD 連接
  ├─ 2. 讀取 369 美股 option chain 數據
  ├─ 3. 計算 CALL/PUT turnover, IV
  ├─ 4. 寫入 Google Sheets (Today) ──── 保留
  ├─ 5. 儲存 CSV 備份 ────────────────── 保留
  ├─ 6. 嘗試從 Notion Historical 讀取昨日數據
  ├─ 7. 計算衍生指標：
  │     P/C ratio, IV spread, Turnover Δ%, IVc Δ%, 異常標記
  ├─ 8. 清除 Daily Snapshot + 寫入今日 369 筆
  └─ 9. 附加 Historical Archive (永久保留)
```

## 使用者每日 Workflow

1. **早上打開 Notion** → `📊 Daily Snapshot`
2. **篩選** → Filter: `Anomaly = 🔴 異常`（一鍵篩出異常股）
3. **排序** → Sort: `Total Turnover` 降序
4. **挑選** → 選定交易標的 → 開倉
5. **記錄** → 在 `📝 Trade Journal` 新增交易紀錄
6. **晚上複盤** → 更新結果 + Lessons Learned
